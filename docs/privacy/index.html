プライバシーポリシー

最終更新: 2026年7月13日

このプライバシーポリシーは、TennisCut日本語版における情報の取り扱いを説明するものです。

1. 取り扱う情報

本アプリは、ユーザーが選択した動画ファイル、編集内容、プロジェクト情報を端末内に保存します。動画の読み込みや保存のために、写真ライブラリへのアクセスを求める場合があります。音声編集を利用する場合は、マイクと音声認識へのアクセスを求める場合があります。

2. 動画データ

読み込んだ動画と書き出した動画は、ユーザーの端末内またはユーザーが選択した保存先に保存されます。開発者がユーザーの動画を直接収集することはありません。

3. 広告

日本語版では、無料利用時にGoogle AdMobのGoogle Mobile Ads SDKを使用します。広告SDKは、広告表示、広告効果測定、不正利用防止、広告配信の改善のために、IPアドレス、端末ID、広告関連データ、クラッシュログ、パフォーマンスデータ、広告やアプリとの操作情報などを取り扱う場合があります。

広告配信事業者による情報の取り扱いについては、Googleのポリシーも確認してください。

4. 購入情報

アプリ内購入とサブスクリプションはAppleによって処理されます。開発者がクレジットカード番号などの支払い情報を取得することはありません。

5. 権限の管理

写真、マイク、音声認識などの権限は、iOSの設定アプリからいつでも変更できます。

6. お問い合わせ

本ポリシーに関するお問い合わせは、tanakashoya919@gmail.com までご連絡ください。

7. ポリシーの変更

本ポリシーは、機能追加、法令対応、App Store要件への対応のために変更される場合があります。
